## Setup Steps

1. Installed Visual Studio Code.
2. Installed the GitHub Copilot extension from the VS Code Marketplace.
3. Signed in with my GitHub account.
4. Activated GitHub Copilot.
5. Installed GitHub Copilot Chat.
6. Installed the YAML extension for YAML validation.
7. Installed the OpenAPI (Swagger) Editor extension for editing and validating OpenAPI specifications.
8. Restarted Visual Studio Code after installation.
9. Verified that Copilot suggestions were working inside the editor.
